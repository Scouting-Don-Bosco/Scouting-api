## Beschrijving

Een NestJS project welke fungeert als backend voor de website van Scouting Don Bosco Heerlerbaan

## Installatie

```bash
$ npm install
```

## Run de applicatie

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
